# DWH_dota2_quality
Обзор проекта

Проект реализует мини-хранилище данных (DWH) на основе открытых данных турниров Dota 2: The International (2011–2021).

Цель проекта — показать полный аналитический пайплайн:
 • загрузка разнородных CSV-файлов
 • очистка и нормализация данных
 • контроль качества данных
 • загрузка в реляционную базу данных (PostgreSQL)
 • построение размерной модели для аналитики

Проект организован по классической архитектуре Stage → DDS → DM.
